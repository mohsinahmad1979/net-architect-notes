# Template The "Checkpoint" Protocol
Before closing a highly productive AI session, or when you notice the AI getting forgetful, issue this exact prompt to the AI. Then take that output and drop it into a file in your 04_AI_Prompts/ folder (e.g., Context-EFCore-Performance-June13.md).

```markdown
We are concluding this session. Act as an expert technical technical writer. Generate a concise markdown summary of our progress, architectural decisions made, code patterns established, and the immediate next steps. Optimize this summary so it can be pasted into a brand new AI chat to completely restore your context.
